
# Shinobi Code App

Frontend for Shinobi Code Extension. This can be accessed either thru [website](https://app.shinobicode.dev/) or VSCode extension.


## Tech Stack

- React
- TailwindCSS
- WASM


## Roadmap

- In V2, we will deprecate the current hand sign detection model and replace it with the model from [this](https://jutsu-ai.vercel.app/) project, as it appears to be actively maintained.


## Run Locally

Clone the project

```bash
  git clone https://github.com/shinobi-c0de/shinobi-code-app.git
```

Go to the project directory

```bash
  cd shinobi-code-app
```

Install dependencies

```bash
  pnpm install
```

Start the server

```bash
  pnpm dev
```
